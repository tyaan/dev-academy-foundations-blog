CSS Positionings: Relative vs Absolute vs Fixed

CSS is a programming language that is used to style web pages.

For example, the color, position and font of text can be altered using CSS.

In this article we will discuss the CSS position property, and how it is used to alter the alignmet of elements on a page.

Relative Position

When we set the position of an element to relative, it means that any position alterations we make will apply relative to its current position.

For example, this text has its position set to relative, and its bottom position set to 10px, so it shifts up 10 pixels from the bottom relative to its normal position.

This same thing can be applied to containers (invisible boxes that hold other elements). For example, lets create 2 containers each holding some text.

Absolute Position

When we set the position of an element to absolute, it means that any position alterations we make will apply relative to its parent element.

For example, this is what happens if we put some text in a container, set the text position to absolute, and set the left position of the text to 10 pixels.

hello

As you can see, the text sits 10 pixels from the left of its parent container. Note that this only works if the parent container has its position explicitly set. If the parent container does not have its position set to anything, then the element will position itself relative to the lowest level parent container that does have its position set.

For example, look at that blue text over there on the left. Look at me! That text has its position set to absolute, and its left position set to 20 pixels. The lowest level parent container of that text that has its position explicitly set is the body of this whole page, so the text sets its left position relative to that.

Fixed Position

When we set the position of an element to fixed, it means that any position alterations we make will apply relative to the viewport (the screen) displaying the page.

You may have noticed that yellow text at the bottom right of your screen, stuck in the same position even as you scroll up and down the page. That text has its position set to fixed, its right position set to 20 pixels, and its bottom position set to 20 pixels. I'm sure you get the point. Its fixed there on your screen. I'm fixed!

Code: Examples;

To set the position of any HTML web page element using CSS, we just need to set position to relative, absolute or fixed, and then specify any or all of top, right, bottom, left.

For example, this CSS code shifts the position of all h3 elements 30 pixels from the top and 50 pixels from the left, relative to their normal position.

h3 {
position: relative;
top: 30px;
left: 50px;
}

This CSS code places the element with id "intro-text" at a fixed position 30 pixels from the top and 50 pixels from the left side of the viewport window.

#intro-text {
position: fixed;
top: 30px;
left: 50px;
}
